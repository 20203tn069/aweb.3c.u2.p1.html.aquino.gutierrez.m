¿Qué es HTML?

Archivo con la extensión de archivo HTM o HTML es un archivo de Lenguaje de marcado de hipertexto. Es el tipo de archivo de página web estándar en Internet.Como los archivos HTML son solo archivos de texto solo contienen texto el que estás leyendo en este momento así como referencias de texto a otros archivos externos, como la imagen en este artículo.

Los archivos HTM y HTML también pueden hacer referencia a otros archivos como, CSS o archivos JS.

    Ventajas

  • Un lenguaje ampliamente utilizado con muchos recursos y una gran comunidad.
  • Se ejecuta de forma nativa en todos los navegadores web.
  • Viene con una curva de aprendizaje plana.
  • Es de código abierto y completamente gratis.
  • Marcado limpio y consistente.
